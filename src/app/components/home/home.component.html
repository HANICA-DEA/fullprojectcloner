What is FullProjectCloner?

FullProjectCloner makes it possible to clone a repository to selected recipients. Cloning will include the full GitHub repository including its wiki and backlog.

How does it work?

  • The user authenticates with GitHub.
  • The user selects a specific GitHub repository.
  • The user chooses recipient(s).
  • A request will be sent out to the recipient(s).
  • The repository will be cloned after the recipient accepted the request.
