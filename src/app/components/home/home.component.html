What is FullProjectCloner?

FullProjectCloner allows the user to clone a repository to someone else, including everything that belongs to it, like a wiki or a backlog.

How does it work?

  1. The user authenticates with GitHub.
  2. The user selects a repository.
  3. The user chooses a recipient.
  4. A request is sent to the recipient.
  5. After the recipient accepts the request, the repository is cloned.
